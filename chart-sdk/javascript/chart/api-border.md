# Api border in JavaScript Chart control

Configures the borders in the chart.

## Properties

### color `string`

The color of the border that accepts value in hex and rgba as a valid CSS color string.

Defaults to *''*

### width `number`

The width of the border in pixels.

Defaults to *1*
