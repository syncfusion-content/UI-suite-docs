# WPF Bullet Graph (SfBulletGraph) Overview

The Bullet Graph is a variation of the bar graph that was developed as a replacement for the dashboard gauges and meters. The bullet graph features a single, primary measure (for example, current year-to-date revenue) and compares that measure to one or more other measures to enrich its meaning (for example, compared to a target). It displays the measures in the context of a qualitative range of performance, such as poor, satisfactory, and good.

## Use cases

* It is used in dashboard environments where screen real estate really counts.
* The bullet graph shows a large amount of data in a very compact space.
* It is handy to visualize data.
* It is used in revenue analysis and expense analysis.
