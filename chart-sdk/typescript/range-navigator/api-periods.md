# Api periods in TypeScript Range navigator control

button settings in period selector

## Properties

### interval `number`

Count value for the button

Defaults to *1*

### intervalType `string`

IntervalType of button

Defaults to *'Years'*

### selected `boolean`

To select the default period

Defaults to *false*

### text `string`

Text to be displayed on the button

Defaults to *null*
