# Welcome to Syncfusion Kanban Component

The Syncfusion Kanban Board is a task management component used to plan, track, and organize work items in a board format. It allows teams to move cards across columns to reflect progress, while maintaining work statuses and priorities. The component supports features such as customizable columns, card templates, swimlane layouts, card editing/CRUD operations, and data binding. It is available across [JavaScript](https://helpstaging.syncfusion.com/gantt-sdk/kanban/javascript/getting-started), [TypeScript](https://helpstaging.syncfusion.com/gantt-sdk/kanban/typescript/getting-started), [Angular](https://helpstaging.syncfusion.com/gantt-sdk/kanban/angular/getting-started), [React](https://helpstaging.syncfusion.com/gantt-sdk/kanban/react/getting-started), [Vue](https://helpstaging.syncfusion.com/gantt-sdk/kanban/vue/getting-started), [ASP.NET MVC](https://helpstaging.syncfusion.com/gantt-sdk/kanban/asp-net-mvc/getting-started), [ASP.NET Core](https://helpstaging.syncfusion.com/gantt-sdk/kanban/asp-net-core/getting-started), [Blazor](https://helpstaging.syncfusion.com/gantt-sdk/kanban/blazor/getting-started-with-web-app), [WPF](https://helpstaging.syncfusion.com/gantt-sdk/kanban/wpf/getting-started), [UWP](https://helpstaging.syncfusion.com/gantt-sdk/kanban/uwp/getting-started), [MAUI](https://helpstaging.syncfusion.com/gantt-sdk/kanban/maui/getting-started) and [WinUI](https://helpstaging.syncfusion.com/gantt-sdk/kanban/winui/getting-started).
The Kanban Board is commonly required in industries where tasks progress through defined stages. It is used in software development for agile workflows, in manufacturing for production tracking, in customer support for ticket resolution, in marketing for campaign progress, in HR for recruitment pipelines, and so on. Across these domains, the Kanban Board provides stage‑based visibility that improves transparency, balances workloads, and supports effective task management.
