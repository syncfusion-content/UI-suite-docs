# Welcome to Syncfusion Gantt Chart Component

The Syncfusion Gantt Chart is a project management component used to visualize and manage work breakdown structures, tasks, dependencies, and resources. It supports planning schedules, tracking progress, and collaboration with features such as task editing, dependency management, timeline customization, and critical path analysis. The component is available across multiple platforms including [JavaScript](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/javascript/es5-getting-started), [TypeScript](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/typescript/getting-started), [Angular](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/angular/getting-started), [React](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/react/getting-started), [Vue](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/vue/getting-started-vue), [ASP.NET MVC](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/asp-net-mvc/getting-started), [ASP.NET Core](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/asp-net-core/getting-started), [Blazor](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/blazor/getting-started-with-web-app), [WPF](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/wpf/getting-started) and [UWP](https://helpstaging.syncfusion.com/gantt-sdk/gantt-chart/uwp/getting-started).
The Gantt Chart is commonly required in industries where projects involve multiple tasks, dependencies, and strict timelines. It is used in construction for phase scheduling, manufacturing for production planning, software development for sprint and release management, event management for organizing activities, healthcare projects for coordinating treatments and resources, enterprise resource planning (ERP) systems for aligning operations, and so on. Across these domains, the Gantt Chart provides a structured timeline view that improves visibility, coordination, and decision‑making.
